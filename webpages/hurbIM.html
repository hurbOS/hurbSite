Hurb's official Instant Messenger. Made with Python and developed by Wilson, Pyrrh0_ and Dreamsmasher. HurbIM is not a client for a mesasging network such as Telegram or Jabber, it will use it's own servers and system. It currently runs off of a terminal text based interface, in which you use commands to control the interface, rather than the classic point & click style. It uses curses, but a gui version is also in development. It will be compatible with most linux systems and there is a plan to port it to Windows at some point. HurbIM is free, open source and encrypted. This means that all of your data is neither used nor abused. Any issues or idea? Feel free to contribute on the github page.
